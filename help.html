How this tracker works

This page explains what the tracker is for, what it covers, how to move your progress around, and a few things worth knowing before you report a bug. This tracker is still a work in progress, so feel free to report bugs or send suggestions.

What This Is

Tracker purpose

  • A checklist for reaching True Perfection in Stardew Valley 1.6.
  • Built to track progress, not to replace the wiki or full game guides.
  • Best for keeping one farm organized, or exporting/importing between farms when needed.
  • Includes a separate hoard mode for long-term item stocking.

How To Use

Basics

  • Use the main tabs to check off fish, cooking, crafting, shipping, and other perfection goals.
  • Use export/import if you want to move your save between browsers or devices.
  • Use the hoard button if you want a separate stash tracker for ingredients, crafting materials, shipping items, and obelisk materials.
  • Use the filters and search bars to narrow what is on screen without changing the actual total counters.

What It Tracks

Included right now

  • Fish collection, cooking recipes, crafting recipes, shipping items, friendships, monster eradication goals, skills, stardrops, golden walnuts, obelisks, and the Gold Clock.
  • Import/export for tracker progress.
  • Hoard mode for shared item needs across cooking, crafting, shipping, and obelisks.
  • Progress summaries on the main tracker and separate progress inside hoard mode.

Pay Attention To

Things to keep in mind

  • Browser progress is stored locally on that device unless you export it.
  • Some item images load from Stardew Valley Wiki URLs, so an internet connection helps them appear.
  • Hoard mode is separate from the regular perfection tracker and does not change the main perfection counters.
  • Hoard mode does not track side costs like bridge repairs, raccoon requests, or house upgrades.
  • Hoard mode tracks exact ingredients only, so processed items like bars are not automatically rolled up from raw materials like ore.

Bug Reports

What helps most

  • Add a screenshot if something looks broken.
  • Mention whether you were on the live site or the local app.
  • Include your browser if the issue happened on the website.
  • Say what tab or mode you were using when it happened.
