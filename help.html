How this tracker works

This page explains what the tracker is for, what it covers, how to move your progress around, and a few things worth knowing before you send feedback. This tracker is still a work in progress, so feel free to report bugs or send suggestions.

What This Is

  • A checklist for reaching True Perfection in Stardew Valley 1.6.
  • Built to track progress, not to replace the wiki or full game guides.
  • Best for keeping one farm organized, or exporting/importing between farms when needed.
  • Includes a separate hoard mode for long-term item stocking.

How To Use

Basics

  • Use the main tabs to check off fish, cooking, crafting, shipping, museum donations, and other perfection goals.
  • Use export/import if you want to move your save between browsers or devices.
  • Use the hoard button if you want a separate stash tracker for ingredients, crafting materials, shipping items, and obelisk materials.
  • Use the filters and search bars to narrow what is on screen without changing the actual total counters.

General

  • Use the General tab as the main dashboard for your farm.
  • Check the overall perfection card first, then use the “left” cards to see what still needs attention.
  • Use the unfinished summary board to jump mentally between fish, cooking, crafting, shipping, and late-game goals.

Fish

  • Use the Fish tab to track every fish needed for perfection.
  • Filter by place, season, weather, and status to narrow the list on screen.
  • Check the box when a fish is caught, or leave it unchecked to keep it in your unfinished list.

Cooking

  • Use the Cooking tab to track cooked recipes and the ingredients needed to finish them.
  • Switch between Recipes, Planner, and Split depending on whether you want the checklist, the ingredient totals, or both.
  • Use the planner inputs to record how many ingredients you already have without marking a recipe cooked too early.

Crafting

  • Use the Crafting tab to check off every crafting recipe needed for perfection.
  • Search or filter by status when you want to focus only on unfinished crafts.
  • The craft list follows the in-game collection order so it is easier to compare with your farm menu.

Shipping

  • Use the Shipping tab to track every item that must be shipped at least once.
  • Use search and status filters to show all items, only shipped items, or only what is still missing.
  • Check an item once it has been shipped so the tab matches your collections progress.

Museum

  • Use the Museum tab to track all 95 artifact and mineral donations in the same order as the museum collections.
  • Check each donation off once it has been turned in so you can see artifact and mineral progress separately.
  • The Museum Donation Reward in Other Perfection turns on automatically once all 95 museum items are checked.

Other Perfection

  • Use this tab for the rest of perfection: friendships, monster goals, skills, stardrops, walnuts, obelisks, and the Gold Clock.
  • Enter counts where the tracker asks for current hearts, levels, kills, or walnuts.
  • Use the checkboxes for finished goals when you have fully completed that requirement.

Dangerous Hoarding Pit

  • Use hoard mode as a separate stash tracker for items needed across cooking, crafting, shipping, and obelisks.
  • Search or filter by category if you want to focus on one kind of stockpile at a time.
  • Use the count inputs and checkboxes there without worrying about changing your regular perfection progress, because hoard mode is its own tracker.

What It Tracks

  • Fish collection, cooking recipes, crafting recipes, shipping items, museum donations, friendships, monster eradication goals, skills, stardrops, golden walnuts, obelisks, and the Gold Clock.
  • Import/export for tracker progress.
  • Hoard mode for shared item needs across cooking, crafting, shipping, and obelisks.
  • Progress summaries on the main tracker and separate progress inside hoard mode.

Pay Attention To

  • Browser progress is stored locally on that device unless you export it.
  • Some item images load from Stardew Valley Wiki URLs, so an internet connection helps them appear.
  • Hoard mode is separate from the regular perfection tracker and does not change the main perfection counters.
  • Hoard mode does not track side costs like bridge repairs, raccoon requests, or house upgrades.
  • Hoard mode tracks exact ingredients only, so processed items like bars are not automatically rolled up from raw materials like ore.

Feedback

  • Add a screenshot if something looks broken.
  • Mention whether you were on the live site or the local app.
  • Include your browser if the issue happened on the website.
  • Say what tab or mode you were using when it happened.
