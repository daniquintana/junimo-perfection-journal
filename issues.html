Current Known Issues

  • GitHub Pages can take a little while to show the newest site update after a push, so a hard refresh may help.
  • Phone and tablet support is still a little temperamental in a few spots, especially around number inputs and fast interactions.
  • Some item images load from Stardew Valley Wiki URLs, so an internet connection helps those appear correctly.

Good To Know

  • Hoard mode tracks exact ingredients only, so processed items like bars are not automatically rolled up from raw materials like ore.
  • Tracker progress is stored locally in your browser unless you export it.
  • The tracker follows the checklist itself, so it does not count extra copies you might want for storage, decorating, or profit.

When To Report Something

  • Report it if the numbers are clearly wrong, something does not save, or a page looks broken.
  • Add a screenshot, your browser, and whether you were on a computer/laptop or a phone/tablet.
  • Mention what tab or mode you were using when it happened.
